# Catalog #

Реализация тестового задания от компании Sherpa Robotics.

## Как запустить ##

* Склонировать репозиторий и перейти в директорию проекта
* Если у вас запущен какой-либо сервер на 80 порту, то необходимо его остановить
* Запустить команду `make` и дождаться конца выполнения скрипта
* Для запуска сервера наберите команду `make server-start`
* Перейти в браузере по адресу http://localhost/
* Документация API http://localhost/api/doc/
* Для остановки сервера наберите команду `make server-stop`
